# Recipe Chatbot

A Python chatbot built using LangChain and Google's Gemini API.

## Features

* Recipe recommendations
* Gemini API integration
* Interactive command-line interface

## Installation

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```env
GOOGLE_API_KEY=your_api_key
```

Run:

```bash
python chatbot.py
```
